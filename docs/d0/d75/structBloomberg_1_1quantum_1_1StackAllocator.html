Detailed Description

template<typename T, unsigned int SIZE>
struct Bloomberg::quantum::StackAllocator< T, SIZE >

Provides fast (quasi zero-time) in-place allocation for STL containers. Types are allocated from a contiguous buffer on the stack. When the buffer is exhausted, allocation is delegated to the heap. The default buffer size is 1000.

Template Parameters
TThe type to allocate.
SIZEThe size of the stack buffer.
Note
This allocator is thread safe. For internal use only.
